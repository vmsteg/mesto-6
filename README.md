# Mesto

Интерактивная страница для публикации фотографий интересных мест. Проект курса «Веб-разработчик» Яндекс Практикума.

## Функциональность

- загрузка карточек и данных пользователя с сервера;
- редактирование профиля и аватара;
- добавление и удаление карточек (с подтверждением);
- постановка и снятие лайков со счётчиком;
- валидация форм;
- модальное окно статистики по клику на логотип.

## Настройка API

Перед запуском укажите в файле `src/scripts/components/api.js` ваш идентификатор группы и личный токен из формы «Токен для проектной работы» в Практикуме.

## Команды

```bash
npm install    # установка зависимостей
npm run dev    # запуск dev-сервера
npm run build  # сборка проекта в dist/
npm run deploy # публикация на GitHub Pages
```
